🚀 HOW TO USE

1. Create a folder named "config" next to VendingMachine.exe:
   dist/
   ├── VendingMachine.exe
   └── config/          ← Create this folder

2. Get your Firebase credentials:
   - Go to Firebase Console
   - Project Settings → Service Accounts
   - Generate New Private Key (Python)
   - Save as "firebase_credentials.json" in config/ folder

3. Double-click VendingMachine.exe
   - App opens at http://127.0.0.1:5000
   - Browser opens automatically
