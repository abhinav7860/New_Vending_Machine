# Quick Reference - Home Screen Sensor Monitoring

## What Was Added?

### ✅ API Endpoint
**URL**: `GET /api/check_sensor_home`
- Polls weight sensors every 10 seconds from the home screen
- Detects when products are taken (weight change > 30g)
- Returns products that were taken with new stock levels

### ✅ Transaction IDs
**Format**: `HSTXN_{timestamp}`
- Example: `HSTXN_1705752145300`
- "HS" = Home Screen
- "TXN" = Transaction
- Distinguishes home-screen detections from normal purchases

### ✅ Stock Updates
- Automatic in-memory updates to database
- Real-time display refresh on home screen
- Visual golden flash animation when stock changes
- Removes "Buy" button when out of stock

### ✅ Logging
- All HSTXN transactions logged to `logs/transactions.json`
- Includes: product ID, name, quantity, price, new stock, timestamp
- Full audit trail for inventory management

## How to Test

1. **Start the app**: Run `python app.py`
2. **Open home screen**: Navigate to `http://localhost:5000`
3. **Open browser console**: Press F12, click "Console" tab
4. **Take a product** from the physical vending machine
5. **Wait up to 10 seconds**
6. **Watch for**:
   - Console message: `[HOME SCREEN] Products detected as taken:`
   - Stock number flashes golden on the product card
   - Stock updates to new quantity

## Behind the Scenes

### When Someone Takes a Product:

```
ESP32 Weight Sensor
        ↓
    [GET] → Read all 4 bins
        ↓
    Compare with last reading
        ↓
    Weight decreased? → YES
        ↓
    >30g change? → YES
        ↓
    Which bin? → Slot 1
        ↓
    Get product info (name, price)
        ↓
    Calculate quantity: 100g ÷ 50g/unit = 2 units
        ↓
    UPDATE stock: 5 → 3
        ↓
    Log: HSTXN_1705752145300
        ↓
    JavaScript detects response
        ↓
    Flash product card + Update display
```

## Console Messages to Look For

✅ **Success Messages**:
- `[HOME SCREEN] Checking weight sensors for product detection...`
- `[HOME SCREEN] Products detected as taken: [...]`
- `[HOME SCREEN STOCK UPDATE] Updated Orange Juice: 8 → 7`

❌ **Error Messages**:
- `[HOME SCREEN SENSOR] Error checking sensors:` → API call failed
- No messages after 10s → Check if sensor is responding

## Configuration

To adjust settings, edit these lines:

### Change polling interval to 5 seconds:
```javascript
// In templates/index.html
setInterval(checkSensorHomeScreen, 5000);  // Change 10000 to 5000
```

### Change weight detection threshold to 20g:
```python
# In app.py
if weight_diff > 20:  # Change 30 to 20
```
